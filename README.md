# 🔤 Hebrew Keyboard Fixer — מתקן מקלדת עברית/אנגלית אוטומטי

תוכנה שרצה ברקע ומתקנת אוטומטית הקלדה בשפה הלא נכונה.

---

## 📦 התקנה (פעם אחת בלבד)

**דרישות מוקדמות:** Python 3.8+ מותקן במחשב.
אם אין לך Python, הורד מ: https://www.python.org/downloads/

### שלבים:

1. חלץ את התיקייה למיקום קבוע (למשל `C:\HebrewKeyboardFixer\`)
2. לחץ פעמיים על **`setup.py`**
3. עקוב אחר ההוראות — ההתקנה אוטומטית
4. בסוף ישאל אם להפעיל עכשיו — לחץ `y`

---

## ▶️ הפעלה יומיומית

לאחר ההתקנה תמצא בשולחן העבודה:
**`Hebrew Keyboard Fixer`** — לחץ פעמיים להפעלה

התוכנה תופיע כאייקון **ירוק עם "HE"** במגש המערכת (ליד השעון בפינה ימנית תחתונה).

---

## ⚙️ אפשרויות (לחיצה ימנית על האייקון)

| אפשרות | הסבר |
|--------|-------|
| **פעיל / מושהה** | הפעל/כבה זמנית |
| **רגישות: 2/3/4 תווים** | כמה תווים לפני שהמערכת מתקנת (ברירת מחדל: 2) |
| **הודעות קופצות** | האם להציג הודעה בכל תיקון |
| **הפעל עם Windows** | הפעל אוטומטית בהפעלת המחשב |
| **יציאה** | סגור את התוכנה |

## 🔍 איך זה עובד?

```
המשתמש מקליד: "שלום"  ← בעברית, אבל המקלדת באנגלית
התוכנה קולטת: "akln"  ← מה שבאמת הוחלק
מזהה: 100% אנגלית-סטייל, המקלדת בעברית
מתקנת: מוחקת "akln", עוברת לעברית, מקלידה "שלום" ✅
```

**המיפוי מדויק 100%** — כי הקשר בין מיקום המקש לאות הוא קבוע.

## ❓ שאלות נפוצות

**ש: התוכנה לא מגיבה**
ת: ודא שהאייקון ירוק במגש המערכת. אם אפור — לחץ עליו ובחר "פעיל".

**ש: התוכנה מתקנת דברים שלא צריך לתקן**
ת: הגדל את הרגישות ל-3 או 4 תווים.

**ש: רוצה שתפעל אוטומטית עם Windows**
ת: לחץ ימני על האייקון → "הפעל עם Windows"

**ש: איך לסגור לגמרי?**
ת: לחץ ימני על האייקון → "יציאה"
